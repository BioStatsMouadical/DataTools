# Correction de l'Exercice 03

## Accès à la correction

La correction de cet exercice est protégée par mot de passe pour vous encourager à essayer par vous-même d'abord.

### Comment utiliser la correction

1. **Essayez d'abord de faire l'exercice par vous-même**
2. **Si vous êtes bloqué, exécutez la correction** :
   ```bash
   cd exercice-03/correction
   python solution.py
   ```
3. **Entrez le mot de passe** lorsque demandé
4. **Comparez votre approche** avec la solution proposée

### Mot de passe

Le mot de passe vous sera demandé lors de l'exécution du script. Contactez le formateur si vous n'avez pas le mot de passe.

**Note** : Cette correction est réservée aux formateurs. Les étudiants qui forkent le projet n'auront pas accès à ce dossier.
